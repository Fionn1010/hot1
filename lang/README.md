# Hill of Tara language packs

These JSON files contain the translated written interface and the eight stop labels for the Hill of Tara tour.

Files:
- `fr.json` — French
- `de.json` — German
- `es.json` — Spanish
- `it.json` — Italian
- `pt.json` — Portuguese (Portugal)
- `hi.json` — Hindi
- `zh.json` — Simplified Chinese

Recommended GitHub location:

`/languages/fr.json`
`/languages/de.json`
`/languages/es.json`
`/languages/it.json`
`/languages/pt.json`
`/languages/hi.json`
`/languages/zh.json`

Each pack contains:
- language and locale metadata
- all main interface labels and messages
- titles, subtitles and primary buttons for all eight stops
- the existing video and GLB filenames for each stop

The current packs translate the written interface. The videos and spoken narration remain in English until translated media is added.
